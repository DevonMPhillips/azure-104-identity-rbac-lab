# azure-104-identity-rbac-lab
This project simulates a small organization using Microsoft Entra ID and Azure RBAC. The goal was to create users, groups, guest accounts, and role assignments while implementing least-privilege access controls.

---
Objective:
Create users and groups

Configure guest accounts

Implement Azure RBAC

Test role inheritance

Configure Self-Service Password Reset

Explore license assignment

---

Enviroment:
Azure Free Tier

Microsoft Entra ID

Azure Portal

Azure CLI
